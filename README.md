# Noughts & Crosses
A simple two-player noughts and crosses game made within Python.

## Features
- Two-player gameplay
- Winning combination detection
- Draw detection
- Correct input management catching errors
- Display of gameboard after every move

## How to run
1. Clone this repository
2. Open it in VS code
3. Run ```bash
python noughts&crosses.py
```

## How to play
- Player 1 is noughts ("O")
- Player 2 is crosses ("X")
- Player 1 places their nought anywhere within the 3 x 3 grid in a position between 0 and 8

0  |  1  |  2
-------------
3  |  4  |  5
-------------
6  |  7  |  8

- Player 2 places their cross and the game continues until a winner or draw happens
- A win occurs when a player gets three of their symbols in a row
